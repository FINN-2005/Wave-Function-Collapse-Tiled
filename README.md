# Wave-Function-Collapse-Tiled

Implementation of a Tiled version of Wave Function Collapse in pygame

## Installation

```bash
git clone https://github.com/FINN-2005/Wave-Function-Collapse-Tiled.git
```
(optional)
```bash
pip install git+https://github.com/FINN-2005/pygame_template.git
```


## Usage

```python
python main.py
```
